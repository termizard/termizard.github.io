# Введение

Добро пожаловать в документацию Termizard. На этой странице кратко описаны проект и быстрый старт.

## О проекте

Termizard — высокопроизводительный эмулятор терминала следующего поколения на Go с GPU‑рендерингом и модульной архитектурой.

## Быстрый старт

1. Склонируйте репозиторий.
2. Установите зависимости.
3. Запустите приложение.

```bash
git clone https://github.com/your/repo.git
cd repo
npm ci
npm run dev
```
