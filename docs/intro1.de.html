Einführung

Willkommen in der Termizard‑Dokumentation. Diese Seite bietet eine kurze Projektübersicht und eine Schnellstart‑Anleitung.

Über das Projekt

Termizard ist ein leistungsstarker Terminal‑Emulator der nächsten Generation, geschrieben in Go mit GPU‑Rendering und modularer Architektur.

Schnellstart

  1. Repository klonen.
  2. Abhängigkeiten installieren.
  3. Anwendung starten.
git clone https://github.com/your/repo.git
cd repo
npm ci
npm run dev
